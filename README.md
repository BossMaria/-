# Расчет ЖКХ для Москвы и Подмосковья

Локальное браузерное приложение для учета до 10 объектов: квартиры, дома, СНТ или иных объектов с разными управляющими компаниями.

## Что считает

- электроэнергию по дневному и ночному счетчику;
- холодную и горячую воду;
- водоотведение как сумму расхода ХВС и ГВС;
- газ;
- ежемесячные членские взносы фиксированной суммой или по площади.

## Квитанции PDF

В разделе `Квитанции` можно загрузить PDF. Программа извлекает текст прямо в браузере и сохраняет разобранные строки начислений в выбранном объекте.

PDF также используется как источник тарифов: если в строках есть ставка, она автоматически добавляется в раздел `Тарифы` с датой начала из месяца документа.

Сейчас поддержаны первые шаблоны:

- ЕПД / квитанции УК формата ПИК: содержание, придомовая территория, ГВС, отопление, электроэнергия МОП, перерасчеты, итог к оплате;
- счет за электроэнергию: Т1 пик, Т2 ночь, Т3 полупик, расход, тариф, начисление, задолженность и итог к оплате.

Для других УК программа сохранит хотя бы общий итог, если сможет найти его в тексте. Новый формат можно добавить как отдельный распознаватель.

## Напоминания

По умолчанию приложение напоминает о внесении данных 20 числа каждого месяца. Оно проверяет все объекты и показывает, где за текущий месяц еще нет показаний или загруженной квитанции.

День можно изменить в блоке `Напоминания`. Кнопка `Уведомления` включает системные уведомления браузера, если браузер и способ открытия сайта это поддерживают. На GitHub Pages это работает надежнее, чем при открытии `index.html` как локального файла.

## Как учитывать разные тарифы

Для каждого объекта тарифы задаются отдельно. Если тариф меняется сезонно или с середины года, добавьте новую строку тарифа с датой начала действия.

Пример:

- `Электроэнергия, день`, 6.43 руб., действует с `2026-01-01` по `2026-06-30`;
- `Электроэнергия, день`, 6.91 руб., действует с `2026-07-01`, без даты окончания.

Расчет за старые месяцы останется по старому тарифу, а новые месяцы будут считаться по новой ставке.

## Как пользоваться

1. Откройте `index.html` в браузере.
2. Добавьте объект: лицевой счет, адрес, УК или СНТ, площадь.
3. В разделе `Услуги` выберите электричество, воду или газ и загрузите PDF с тарифами.
4. Если по воде у УК две квитанции, загрузите обе в раздел `Вода`: тарифы объединятся в справочник объекта.
5. В разделе `Показания` добавляйте ежемесячные данные счетчиков.
6. В разделе `Расчет` выберите месяц и нажмите `Рассчитать`.

Данные сохраняются в браузере. Для резервной копии используйте `Экспорт`, для восстановления — `Импорт`.
