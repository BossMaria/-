# Расчет ЖКХ для Москвы и Подмосковья

Локальное браузерное приложение для учета до 10 объектов: квартиры, дома, СНТ или иных объектов с разными управляющими компаниями.

## Что считает

- электроэнергию по дневному и ночному счетчику;
- холодную и горячую воду;
- водоотведение как сумму расхода ХВС и ГВС;
- газ;
- ежемесячные членские взносы фиксированной суммой или по площади.

## Как учитывать разные тарифы

Для каждого объекта тарифы задаются отдельно. Если тариф меняется сезонно или с середины года, добавьте новую строку тарифа с датой начала действия.

Пример:

- `Электроэнергия, день`, 6.43 руб., действует с `2026-01-01` по `2026-06-30`;
- `Электроэнергия, день`, 6.91 руб., действует с `2026-07-01`, без даты окончания.

Расчет за старые месяцы останется по старому тарифу, а новые месяцы будут считаться по новой ставке.

## Как пользоваться

1. Откройте `index.html` в браузере.
2. Добавьте объект и укажите регион, УК или СНТ.
3. В разделе `Тарифы` внесите тарифы для этого объекта.
4. В разделе `Показания` добавляйте ежемесячные данные счетчиков.
5. В разделе `Расчет` выберите месяц и нажмите `Рассчитать`.

Данные сохраняются в браузере. Для резервной копии используйте `Экспорт`, для восстановления — `Импорт`.
